# Customer Management API (Serverless, AWS)

## Description
Serverless backend for managing customers, built with **TypeScript**, **NestJS**, and **AWS**.

## Serverless
- AWS Lambda
- API Gateway
- DynamoDB

## Deployment
- Serverless Framework
- IAM
- CloudWatch

## Stack
- Node.js
- TypeScript
- NestJS

## Install & Run
```bash
git clone https://github.com/your-username/customer-serverless.git
cd customer-serverless
npm install
npm run start:dev

Example API Calls

GET /customers – Get all customers

POST /customers – Create a new customer

PUT /customers/{id} – Update a customer

DELETE /customers/{id} – Delete a customer

```

Note

Demo project to practice serverless backend deployment; includes automated deployments and CloudWatch logging.
